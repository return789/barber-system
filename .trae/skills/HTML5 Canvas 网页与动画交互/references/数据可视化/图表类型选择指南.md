# 图表类型选择指南

## 如何选择合适的图表？

选择正确的图表类型是有效数据可视化的关键。本指南帮助你根据数据特性和展示目标做出最佳选择。

## 图表类型速查表

| 数据关系 | 推荐图表 | 适用场景 |
|---------|---------|---------|
| 趋势/变化 | 折线图、面积图 | 时间序列、趋势分析 |
| 比较 | 柱状图、条形图 | 类别对比、排名 |
| 占比 | 饼图、环形图、堆叠图 | 份额、分布 |
| 相关性 | 散点图、气泡图 | 变量关系、聚类 |
| 分布 | 直方图、箱线图 | 统计分布、异常值 |
| 层次 | 树状图、旭日图 | 分类层级、占比 |
| 多维 | 雷达图、平行坐标 | 性能评估、对比 |
| 地理 | 地图、热力地图 | 区域数据、分布 |
| 流动 | 桑基图、弦图 | 流量、关系 |
| 网络 | 力导向图、节点图 | 社交网络、拓扑 |

## 详细图表说明

### 1. 折线图（Line Chart）

**用途**：展示数据随时间的变化趋势

**优点**：
- 清晰展示趋势和变化速率
- 适合多系列对比
- 易于理解

**缺点**：
- 不适合大量类别
- 数据点过多时拥挤

**Canvas 实现要点**：
```javascript
function drawLineChart(ctx, data, options) {
  const { width, height, padding } = options;
  const chartWidth = width - padding * 2;
  const chartHeight = height - padding * 2;
  
  // 计算比例
  const xScale = chartWidth / (data.length - 1);
  const yRange = getMax(data) - getMin(data);
  const yScale = chartHeight / yRange;
  
  // 绘制线条
  ctx.beginPath();
  ctx.strokeStyle = '#0066ff';
  ctx.lineWidth = 2;
  
  data.forEach((point, i) => {
    const x = padding + i * xScale;
    const y = height - padding - (point.value - getMin(data)) * yScale;
    
    if (i === 0) {
      ctx.moveTo(x, y);
    } else {
      ctx.lineTo(x, y);
    }
  });
  
  ctx.stroke();
  
  // 绘制数据点
  data.forEach((point, i) => {
    const x = padding + i * xScale;
    const y = height - padding - (point.value - getMin(data)) * yScale;
    
    ctx.beginPath();
    ctx.fillStyle = '#ffffff';
    ctx.arc(x, y, 4, 0, Math.PI * 2);
    ctx.fill();
    
    ctx.strokeStyle = '#0066ff';
    ctx.lineWidth = 2;
    ctx.stroke();
  });
}
```

**最佳实践**：
- 限制系列数量（≤ 5）
- 使用不同颜色/线型区分系列
- 添加数据标签（关键点）
- Y 轴从 0 开始（除非有特殊理由）

### 2. 面积图（Area Chart）

**用途**：强调数量累积和占比

**变体**：
- 堆叠面积图（展示总量）
- 百分比面积图（展示占比）
- 流图（对称美观）

**Canvas 实现要点**：
```javascript
function drawAreaChart(ctx, data, options) {
  const gradient = ctx.createLinearGradient(0, 0, 0, height);
  gradient.addColorStop(0, 'rgba(0, 102, 255, 0.6)');
  gradient.addColorStop(1, 'rgba(0, 102, 255, 0.05)');
  
  ctx.beginPath();
  ctx.fillStyle = gradient;
  
  // 绘制面积区域
  data.forEach((point, i) => {
    const x = padding + i * xScale;
    const y = height - padding - point.value * yScale;
    
    if (i === 0) {
      ctx.moveTo(x, y);
    } else {
      ctx.lineTo(x, y);
    }
  });
  
  // 闭合路径
  ctx.lineTo(padding + (data.length - 1) * xScale, height - padding);
  ctx.lineTo(padding, height - padding);
  ctx.closePath();
  ctx.fill();
}
```

### 3. 柱状图（Bar Chart）

**用途**：类别间数值对比

**变体**：
- 垂直柱状图
- 水平条形图（类别名长时）
- 堆叠柱状图（展示组成）
- 分组柱状图（多系列对比）

**Canvas 实现要点**：
```javascript
function drawBarChart(ctx, data, options) {
  const barWidth = chartWidth / data.length * 0.7;
  const gap = chartWidth / data.length * 0.3;
  
  data.forEach((item, i) => {
    const x = padding + i * (barWidth + gap) + gap / 2;
    const barHeight = item.value * yScale;
    const y = height - padding - barHeight;
    
    // 绘制柱状图
    const gradient = ctx.createLinearGradient(x, y, x, height - padding);
    gradient.addColorStop(0, item.color || '#0066ff');
    gradient.addColorStop(1, item.color || '#0066ff' + '80');
    
    ctx.fillStyle = gradient;
    ctx.fillRect(x, y, barWidth, barHeight);
    
    // 顶部圆角
    ctx.beginPath();
    ctx.fillStyle = item.color || '#0066ff';
    ctx.arc(x + barWidth / 2, y, barWidth / 2, Math.PI, 0);
    ctx.fill();
    
    // 数值标签
    ctx.fillStyle = '#333333';
    ctx.font = '12px sans-serif';
    ctx.textAlign = 'center';
    ctx.fillText(item.value, x + barWidth / 2, y - 5);
  });
}
```

**最佳实践**：
- 类别 ≤ 15 时使用垂直柱状图
- 类别名长时使用水平条形图
- 堆叠层数 ≤ 5
- 添加数值标签提高可读性

### 4. 饼图/环形图（Pie/Donut Chart）

**用途**：展示部分与整体的关系

**优点**：
- 直观展示占比
- 易于理解

**缺点**：
- 不适合多类别（≤ 7）
- 精确对比困难
- 3D/爆炸效果降低可读性

**Canvas 实现要点**：
```javascript
function drawPieChart(ctx, data, options) {
  const total = data.reduce((sum, item) => sum + item.value, 0);
  const centerX = width / 2;
  const centerY = height / 2;
  const radius = Math.min(width, height) / 2 - padding;
  
  let startAngle = -Math.PI / 2;
  
  data.forEach((item) => {
    const sliceAngle = (item.value / total) * Math.PI * 2;
    
    ctx.beginPath();
    ctx.fillStyle = item.color;
    ctx.moveTo(centerX, centerY);
    ctx.arc(centerX, centerY, radius, startAngle, startAngle + sliceAngle);
    ctx.closePath();
    ctx.fill();
    
    // 边框
    ctx.strokeStyle = '#ffffff';
    ctx.lineWidth = 2;
    ctx.stroke();
    
    startAngle += sliceAngle;
  });
  
  // 环形图：绘制中心圆
  if (options.type === 'donut') {
    ctx.beginPath();
    ctx.fillStyle = '#ffffff';
    ctx.arc(centerX, centerY, radius * 0.6, 0, Math.PI * 2);
    ctx.fill();
    
    // 中心文字
    ctx.fillStyle = '#333333';
    ctx.font = 'bold 16px sans-serif';
    ctx.textAlign = 'center';
    ctx.fillText(total, centerX, centerY + 5);
  }
}
```

**最佳实践**：
- 类别 ≤ 7
- 按大小排序
- 添加百分比标签
- 优先使用环形图（可展示总计）

### 5. 散点图（Scatter Plot）

**用途**：展示两个变量的关系

**优点**：
- 发现相关性和聚类
- 适合大数据集

**Canvas 实现要点**：
```javascript
function drawScatterPlot(ctx, data, options) {
  data.forEach((point) => {
    const x = padding + point.x * xScale;
    const y = height - padding - point.y * yScale;
    
    ctx.beginPath();
    ctx.fillStyle = point.color || 'rgba(0, 102, 255, 0.6)';
    ctx.arc(x, y, point.size || 5, 0, Math.PI * 2);
    ctx.fill();
    
    // 边框
    ctx.strokeStyle = 'rgba(255, 255, 255, 0.8)';
    ctx.lineWidth = 1;
    ctx.stroke();
  });
  
  // 趋势线（可选）
  if (options.trendLine) {
    const { slope, intercept } = linearRegression(data);
    ctx.beginPath();
    ctx.strokeStyle = '#ff0000';
    ctx.lineWidth = 2;
    ctx.setLineDash([5, 5]);
    
    const x1 = 0;
    const y1 = intercept;
    const x2 = 1;
    const y2 = slope + intercept;
    
    ctx.moveTo(padding + x1 * xScale, height - padding - y1 * yScale);
    ctx.lineTo(padding + x2 * xScale, height - padding - y2 * yScale);
    ctx.stroke();
    ctx.setLineDash([]);
  }
}
```

### 6. 气泡图（Bubble Chart）

**用途**：散点图扩展，第三维度用大小表示

**Canvas 实现要点**：
- 在散点图基础上添加 size 映射
- 使用半透明颜色避免遮挡
- 添加标签（大泡泡）

### 7. 雷达图（Radar Chart）

**用途**：多维度数据对比

**Canvas 实现要点**：
```javascript
function drawRadarChart(ctx, data, options) {
  const centerX = width / 2;
  const centerY = height / 2;
  const radius = Math.min(width, height) / 2 - padding;
  const levels = 5;
  const angleStep = (Math.PI * 2) / data.length;
  
  // 绘制网格
  for (let level = 1; level <= levels; level++) {
    const r = (radius / levels) * level;
    
    ctx.beginPath();
    ctx.strokeStyle = '#e0e0e0';
    ctx.lineWidth = 1;
    
    for (let i = 0; i <= data.length; i++) {
      const angle = angleStep * i - Math.PI / 2;
      const x = centerX + Math.cos(angle) * r;
      const y = centerY + Math.sin(angle) * r;
      
      if (i === 0) {
        ctx.moveTo(x, y);
      } else {
        ctx.lineTo(x, y);
      }
    }
    
    ctx.stroke();
  }
  
  // 绘制数据
  data.series.forEach((series) => {
    ctx.beginPath();
    ctx.fillStyle = series.fillColor;
    ctx.strokeStyle = series.color;
    ctx.lineWidth = 2;
    
    series.values.forEach((value, i) => {
      const angle = angleStep * i - Math.PI / 2;
      const r = (value / maxValue) * radius;
      const x = centerX + Math.cos(angle) * r;
      const y = centerY + Math.sin(angle) * r;
      
      if (i === 0) {
        ctx.moveTo(x, y);
      } else {
        ctx.lineTo(x, y);
      }
    });
    
    ctx.closePath();
    ctx.fill();
    ctx.stroke();
  });
}
```

**最佳实践**：
- 维度数量 3-8
- 使用相同量纲
- 系列数量 ≤ 3
- 添加网格线提高可读性

### 8. 热力图（Heatmap）

**用途**：展示数据密度或强度

**Canvas 实现要点**：
```javascript
function drawHeatmap(ctx, data, options) {
  const cellWidth = chartWidth / cols;
  const cellHeight = chartHeight / rows;
  
  data.forEach((row, y) => {
    row.forEach((value, x) => {
      const color = getColorFromScale(value, options.colorScale);
      
      ctx.fillStyle = color;
      ctx.fillRect(
        padding + x * cellWidth,
        padding + y * cellHeight,
        cellWidth - 1,
        cellHeight - 1
      );
      
      // 数值标签
      if (options.showLabels) {
        ctx.fillStyle = getContrastColor(color);
        ctx.font = '12px sans-serif';
        ctx.textAlign = 'center';
        ctx.fillText(
          value,
          padding + x * cellWidth + cellWidth / 2,
          padding + y * cellHeight + cellHeight / 2 + 4
        );
      }
    });
  });
}
```

### 9. 树状图（Treemap）

**用途**：展示层级数据和占比

**Canvas 实现要点**：
```javascript
function drawTreemap(ctx, data, x, y, width, height) {
  // 简单切片算法
  const total = data.reduce((sum, item) => sum + item.value, 0);
  
  data.forEach((item) => {
    const area = (item.value / total) * width * height;
    const itemWidth = width;
    const itemHeight = area / itemWidth;
    
    ctx.fillStyle = item.color;
    ctx.fillRect(x, y, itemWidth, itemHeight);
    
    ctx.strokeStyle = '#ffffff';
    ctx.lineWidth = 2;
    ctx.strokeRect(x, y, itemWidth, itemHeight);
    
    // 标签
    if (itemHeight > 20) {
      ctx.fillStyle = '#ffffff';
      ctx.font = '12px sans-serif';
      ctx.fillText(item.name, x + 5, y + 15);
    }
    
    y += itemHeight;
  });
}
```

## 色彩与数据可视化

### 色彩类型
- **分类色彩**：不同颜色表示不同类别（离散）
- **顺序色彩**：单色渐变表示大小（连续）
- **差异色彩**：双色渐变表示正负（连续）

### 推荐配色方案

#### 分类色彩（6-8 色）

### 色彩最佳实践
- 分类数据：使用色相差异
- 顺序数据：使用明度/饱和度梯度
- 差异数据：使用双色渐变，中点为中性色
- 色盲友好：避免红绿搭配，使用蓝橙
- 深色背景：提高饱和度，降低明度

## 图表设计原则

1. **准确传达**：不误导，不失真
2. **简洁明了**：去除多余装饰
3. **一致性**：相同数据用相同视觉编码
4. **对比度**：确保可读性
5. **标签清晰**：文字大小合适
6. **交互增强**：悬停显示详情
7. **响应式**：适配不同屏幕

## 常见错误

❌ 3D 饼图（扭曲比例）  
❌ 过多类别（> 15）  
❌ 无标签/图例  
❌ Y 轴不从 0（柱状图）  
❌ 颜色过多难以区分  
❌ 装饰过度影响阅读  

## 交互式增强

### 悬停提示
```javascript
canvas.addEventListener('mousemove', (e) => {
  const rect = canvas.getBoundingClientRect();
  const mouseX = e.clientX - rect.left;
  const mouseY = e.clientY - rect.top;
  
  // 命中检测
  const hoveredElement = findElementAtPosition(mouseX, mouseY);
  
  if (hoveredElement) {
    showTooltip(hoveredElement, mouseX, mouseY);
  } else {
    hideTooltip();
  }
});
```

### 缩放与平移
```javascript
canvas.addEventListener('wheel', (e) => {
  e.preventDefault();
  const zoomFactor = e.deltaY > 0 ? 0.9 : 1.1;
  zoomChart(zoomFactor);
  render();
});

let isPanning = false;
let lastX, lastY;

canvas.addEventListener('mousedown', (e) => {
  isPanning = true;
  lastX = e.clientX;
  lastY = e.clientY;
});

canvas.addEventListener('mousemove', (e) => {
  if (isPanning) {
    const dx = e.clientX - lastX;
    const dy = e.clientY - lastY;
    panChart(dx, dy);
    lastX = e.clientX;
    lastY = e.clientY;
    render();
  }
});

canvas.addEventListener('mouseup', () => {
  isPanning = false;
});
```

## 性能提示

- 大量数据点时使用 Canvas 而非 SVG
- 使用降采样（> 10000 点）
- 离屏 Canvas 缓存静态元素
- 视口裁剪不可见元素
- 批量绘制相同样式元素
