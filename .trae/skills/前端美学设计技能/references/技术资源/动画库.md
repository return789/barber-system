# 动画库

本文档推荐适合不同场景的动画库。

## 推荐动画库

### 1. Motion (Framer Motion v11+)

**推荐场景**: React 项目，声明式动画

```bash
npm install motion
```

```jsx
import { motion } from 'motion';

function Button({ children }) {
  return (
    <motion.button
      whileHover={{ scale: 1.05 }}
      whileTap={{ scale: 0.95 }}
      initial={{ opacity: 0 }}
      animate={{ opacity: 1 }}
      transition={{ duration: 0.3 }}
    >
      {children}
    </motion.button>
  );
}

// 列表动画
function List({ items }) {
  return (
    <motion.ul>
      {items.map((item, i) => (
        <motion.li
          key={item.id}
          initial={{ opacity: 0, y: 20 }}
          animate={{ opacity: 1, y: 0 }}
          transition={{ delay: i * 0.1 }}
        >
          {item.text}
        </motion.li>
      ))}
    </motion.ul>
  );
}
```

### 2. GSAP (GreenSock)

**推荐场景**: 复杂时间线动画，专业级性能

```bash
npm install gsap
```

```javascript
import gsap from 'gsap';

// 基础动画
gsap.to('.element', {
  x: 100,
  duration: 1,
  ease: 'power2.out'
});

// 时间线
const tl = gsap.timeline();

tl.from('.title', { y: 50, opacity: 0 })
  .from('.subtitle', { y: 30, opacity: 0 }, '-=0.5')
  .from('.button', { scale: 0.8, opacity: 0 }, '-=0.3');
```

### 3. Anime.js

**推荐场景**: 轻量级通用动画，Vue/原生 JS 项目

```bash
npm install animejs
```

```javascript
import anime from 'animejs';

// 目标选择器
anime({
  targets: '.card',
  translateY: [-20, 0],
  opacity: [0, 1],
  delay: anime.stagger(100),
  easing: 'easeOutExpo'
});
```

### 4. Lottie

**推荐场景**: 复杂 AE 动画，图标动画，loading 动画

```bash
npm install lottie-web
```

```javascript
import lottie from 'lottie-web';

const animation = lottie.loadAnimation({
  container: document.getElementById('lottie-container'),
  renderer: 'svg',
  loop: true,
  autoplay: true,
  path: 'animation.json'
});
```

### 5. @jagannath8/react-text-reveal

**推荐场景**: 文字揭示动画

```jsx
import TextReveal from '@jagannath8/react-text-reveal';

function Title() {
  return (
    <TextReveal
      text="Hello World"
      animation={{ y: 100, opacity: 0 }}
      reveal={300}
      easing="power2.out"
    />
  );
}
```

## 性能考虑

### 何时使用动画库
- 复杂的、时间线编排的动画
- 需要精确控制 easing 和 duration
- 需要暂停、倒放等功能
- 需要 SVG 路径动画

### 何时使用 CSS
- 简单的 hover 效果
- 单次触发的 transition
- 性能关键的动画
- loading 状态

### 性能检查清单
- [ ] 使用 `transform` 和 `opacity`
- [ ] 避免动画 `width`、`height`、`top`、`left`
- [ ] 使用 `will-change` 谨慎
- [ ] 考虑 `prefers-reduced-motion`
- [ ] 测试移动端性能

```css
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```
