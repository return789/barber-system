# 配色方案

本文档提供独特配色方案指导，避免陈词滥调的 AI 美学。

## 核心原则

### 禁止的通用配色
- ❌ 紫色渐变 (#8B5CF6 → #A855F7) on 白色背景
- ❌ 蓝色渐变 (Blue → Cyan)
- ❌ 过度使用 #6366F1 (Indigo)
- ❌ system fonts 默认色板

### 应该追求的
- ✅ dominant 单一主色 + 锐利点缀
- ✅ 意想不到的组合
- ✅ 与整体美学高度一致
- ✅ 有故事/情感连接的色彩

## 高级技巧

### 1. 使用 HSL 而非 HEX
```css
/* 更易于创建色彩变体 */
--color-primary: hsl(210, 100%, 50%);
--color-primary-light: hsl(210, 100%, 70%);
--color-primary-dark: hsl(210, 100%, 30%);
```

### 2. 创建色彩节奏
```css
/* 主色占据 60%，次色 30%，点缀 10% */
.color-dominant { background: var(--color-primary); }
.color-secondary { background: var(--color-secondary); }
.color-accent { background: var(--color-accent); }
```

### 3. 添加纹理色彩
```css
/* 不仅仅使用纯色 */
background: linear-gradient(
  135deg,
  var(--color-primary) 0%,
  color-mix(in srgb, var(--color-primary) 80%, var(--color-secondary)) 100%
);
```

### 4. 暗色模式映射
```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-background: #0D0D0D;
    --color-text: #E8E8E8;
    --color-primary: #4A9EFF;
  }
}
```
